# WhatsTux

## Minimalistic WhatsApp Destkop Emulator for Linux

> Lets you run the [web version of WhatsApp](https://web.whatsapp.com) on you local machine 


---
### To install run:

````
npm install
````

### Run project in dev:
````
npm run tauri dev
````

### Run project in build:
````
npm run tauri build
````
